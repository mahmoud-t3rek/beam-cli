# Services  

## Prompt 
This library handles the core domain behaviors and API listings.

### Core Responsibilities:
1.  **Behavior Execution**: Use `js-behaviours` to run specific logic units.
2.  **Listing**: Provide functionality to list available APIs or Behaviors.
3.  **Verification**: Verify login status and API existence.

### References:
- Uses: `js-behaviours` package.
